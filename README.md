# 🖊️ CodePen Clone – My First React Project

A simple yet powerful CodePen-like playground built using React and Vite.  
This was my first ever React project — full of bugs, learning, trial and error, and finally... joy 🎉

## ✨ Features

- 🔤 Live HTML, CSS, and JavaScript editors (powered by CodeMirror)
- ⚡ Real-time preview with auto-refresh
- 🌙 Light/Dark mode toggle
- 💾 Save code to `localStorage` (so your work stays safe!)
- 📥 Download code as a `.zip` file (HTML, CSS, JS)

> Upcoming features:
> - 💾 Auto save + load on tab reopen  
> - 📁 Support for user-created templates  
> - 🧪 Live JS error console  
> - 📤 Deployment to GitHub Pages

## 🛠️ Tech Stack

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [CodeMirror](https://codemirror.net/)
- [JSZip](https://stuk.github.io/jszip/)
- [FileSaver.js](https://github.com/eligrey/FileSaver.js)
